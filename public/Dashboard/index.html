Information Page

The Credit Card Fraud Detection System is a web-based application for detecting fraudulent transactions.

This Project uses a Machine Learning model to identify whether a transaction is fraudulent or not.

This Project Uses the following Backend Technologies:

  • Machine Learning model written in Python.
  • FastAPI to make a API and connect it with the ML Model
  • Heroku PaaS (Platform as a Service) | used to host FastAPI and ML Model
  • Firebase BaaS (Backend as a Service) | used to host this Webpage & add Functionality like Authentication and Database

This Project Uses the following Frontend Technologies:

  • Firebase used to host this Webpage
  • Languages: HTML, CSS, Javascript and jQuery to fetch API.
  • Tailwind CSS a CSS framework used to style this Webpage.
  • Flowbite Tailwind CSS component library used to add Day/Knight Mode.
  • Heroicons SVG icons
  • Heroku PaaS (Platform as a Service) | used to host FastAPI and ML Model

The Machine Learning Model is trained using the DATASET provided by kaggle.com.

V1, V2 as described by kaggle.com the DataSet provider for this project are the numerical input variables which are the result of a PCA transformation.

i.e to keep the identity of people private, due to confidentiality issues, they cannot provide the original features and more background information about the data. Features V1, V2, … V28 are the principal components obtained with PCA.

Also note that the DataSet used in the training of the ML Model is highly unbalanced..
